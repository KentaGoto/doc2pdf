# doc2pdf
Generate pdf from doc and docx.

## HOW TO USE

Terminal

```
> doc2pdf.py
Dir: <Drag and drop the folder containing doc(x).>
Processing...
...
...
...
Done!
Enter to exit.
```

**NOTE:**

- After processing is complete, a copy (\_\_copy) folder is created. The PDF(s) will be generated in that folder.

## REQUIRES

- Windows
- Word 32bit
